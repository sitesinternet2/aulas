Comandos Git

clonando

inicialização das atividades:

  • git fetch --all

    sicroniza o repositorio local com o reposito remoto

  • git pull

    faz o download das atualizações do servidor remoto

finalização das atividades:

  • git status

    lista as modificações

  • git add .

    adiciona as modificações mo servidor local

  • git commit -m "[texto]"

    monta as modificações para envio ao servidor remoto

  • git push

    faz upload das modificações do servidor local para p servidor remoto
