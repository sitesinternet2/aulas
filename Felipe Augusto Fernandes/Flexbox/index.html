flexbox
É usado para organizar os itens dentro de um elemento principal.
flex-direction
É utilizado para definir o fluxo dos elementos a serem exibidos
flex-wrap
Se for necessario uma quebra de linha e usado ele para separar as divs
justify-content
É usado para definir o alinhamento dos itens ao longo do eixo principal
align-content
Define como as linhas seram distribuidas pelo eixo transversal do container
align-items
Define como os itens sao distribuidos pelo eixo transversal do container
Order
Serve para alterar a order que os itens foram distribuidos no container
flex-shrink
Define a proporção que um item deve diminuir caso necessario
flex-grow
Define a proporção que um item deve Crescer caso necessario
flex-basis
Define o tamanho inicial que um item deve ter antes que o espaço ao seu redor seja distribuido
align-self
Permite sobrescrever o comportamento que foi definido pelo align-items
