Comandos Git

inicilização das atividades

  • Git fetch --all
    sincroniza o repositorio local com o repositorio remoto
  • Git pull
    faz o dowload das atualizacoes do servidor remoto

finalização das atividades

  • git status
    lista as modificaçoes
  • git add .
    adiciona as modificaçoes no servidor local
  • git commit -m ""
    monta as modificaçoes para
  • git push
