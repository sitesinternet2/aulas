Comandos Git

inicialização das atividades

clonando

  • git clone

    clona a pasta do github para a sua maquina.

  • git fetch --all

    sincroniza os repositorios repomotos.

  • git pull

    faz o download das atualizações no servidor remoto.

finalização das atividades

  • git status

    mostra oque tem de diferente do servidor remoto.

  • git add .

    adiciona as modificações do servidor do servidor local.

  • git commit -m "texto"

    adiciona uma commit no github com o texto que você quizer.

  • git push

    faz o upload dos novos arquivos para o servidor remoto.
