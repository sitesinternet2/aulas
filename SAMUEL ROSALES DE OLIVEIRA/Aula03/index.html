Comandos Git

Inicialização das Atividade

  • git clone [link Github] (Clona o repositorio do servidor remoto para o servior local)

Sincronia de Arquivos

  • git fetch --all (Sincroniza o repositorio local com o remoto)
  • git pull (Faz o download das autolizações do servidor remoto)

Finalização das Atividade

  • git status (Lista as modificações)
  • git add . (Prepara as modificações para o servidor local)
  • git commit -m "[text]"(Monta e nomeia as modificações para o servidor remoto)
  • git push (Faz uploud dos commit do servidor local para o servidor remoto)
